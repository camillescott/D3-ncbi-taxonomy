Interactive NCBI Taxonomic Tree

Camille Scott

This visualization is an interactive version of the complete NCBI taxonomic tree. The code is a bit of a mashup of several dendrogram examples; the result is a collapsible circular layout, with nodes sized based on the number of children. flask is used as a webserver to drive it, which unfortuantely makes it difficult to deliver is a self-contained page; instead, I will host it on one of my heroku instances.

The root node is colored red, while the currently selected node is colored green. The user can select a node either by clicking on it, or by using the w, a, s, and d keys to move between parent/children and siblings. The selected node's children can be expanded with the arrow keys: left and right to collapse and expand 10 children at a time, and up and down to expand or collapse all children.
